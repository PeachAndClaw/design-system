# Peach & Claw — Design System

Explorations et fondations du design system partagé de Sylvain & Gianna.

La page publiée présente la direction unifiée retenue :

- 60 % Atelier nocturne
- 30 % Velours éditorial
- 10 % Couture tranchante

Les trois explorations initiales restent disponibles dans `explorations-v1.html`.
La V2 est une direction validée, pas encore la charte finale complète.

## Identité

`approval.html` présente uniquement la famille approuvée : emblème historique,
monogramme Fusion enveloppante et logotype B vectorisé. `identity.html` conserve
les explorations antérieures. Les règles sont documentées dans `BRAND-GUIDELINES.md`.

La vectorisation est reproductible avec :

```sh
npm install
npm run build:logo
```
